# Pixel Canvas（像素画纸）模组说明

**版本**：NeoForge 1.21.1  
**Mod ID**：`paintingmod`  
**测试版功能**：手持画纸 + 贴墙画纸 + 像素绘制 + 多人实时同步

---

## 1. 模组是做什么的？

Pixel Canvas 让你在《我的世界》里像用一张真正的画纸一样作画：

- **手持画纸**：拿在手上右键空气即可打开绘制界面，画好的内容会保存在这张纸里。
- **贴墙画纸**：手持画纸右键任意方块表面，会把这张纸贴到该面上，其他玩家实时看见你作画。
- **取回画纸**：Shift + 右键墙上的画纸，会把它取回成物品，当前画面同步回你手里。

画纸是**薄薄的一层**（约 2 像素厚），可以贴在地面、墙壁、天花板等任意坚固方块表面上，不会像实心方块那样违和。

---

## 2. 怎么获得画纸？

### 创造模式
在创造物品栏的 **“Pixel Paint”** 标签页里找到：
- **画纸（Paint Paper）**
- **万能染料（Universal Dye）**

### 生存模式合成

```
画纸：8 张纸围一圈，中间 1 根木棍  =>  4 张画纸
万能染料：红染料 + 绿染料 + 蓝染料（无序合成）
```

---

## 3. 怎么画画？

### 手持作画
1. 拿起一张画纸。
2. 右键空气，打开绘制界面。
3. 画完后关闭界面，内容会自动存进这张纸里。

### 贴墙作画
1. 拿起一张画纸（里面可以有预先画好的内容）。
2. 对准任意方块面右键，画纸会贴到那个面上。
3. 右键墙上的画纸继续作画，附近玩家实时看见笔触。
4. Shift + 右键墙上的画纸可以取回，画面同步到手里的画纸。

---

## 4. 绘制界面

- **左侧**：16 种原版染料色块（直接显示原版染料图标）。点击选色，或开启“混合”后累加多种染料。
- **中间**：画布区域，默认 **128×128** 像素。高分辨率画纸在界面里会统一缩放显示，不会比小分辨率还小。
- **右侧**：工具按钮（画笔、橡皮、取色、填充、撤销、重做、清空、保存、调整尺寸）。
- **底部**：混合控制、万能染料 RGB 滑块（需背包有万能染料）、状态信息。

绘制体验：
- 鼠标按下 → 拖动 → 松开为**一笔**。
- 落笔后客户端先本地即时显示，每 50ms 批量同步一次，松手再补发完整数据，所以**颜色跟手**。
- 如果服务端开启染料消耗，每一笔会按当前颜料来源扣除背包里对应的染料（基础染料/混合染料/万能染料各 1 个）。

---

## 5. 多人联机

- 绘制时笔触会**实时广播**给附近 16 格内的玩家，别人不用重进就能看到。
- 服务端会校验距离（< 8 格）和权限，陌生人无法乱涂。
- 染料不足时本笔会被回滚，并提示“染料不足”。

---

## 6. 配置

配置文件在 `config/paintingmod-server.toml`：
- `consumeDye`：是否消耗染料（默认 true）。
- `maxCanvasSize`：最大分辨率，上限 128（默认 128）。
- `defaultCanvasSize`：新建画纸默认分辨率（默认 128）。
- `universalDyeEnabled`：是否启用万能染料（默认 true）。

---

## 7. 已知限制（测试版）

- 暂不支持把画导出为 PNG 图片（M5 计划）。
- 手持画纸和墙上画纸在贴出后是独立编辑的；取回时才会同步回手里。
- **AI 识别功能尚未接入**：这是 M6 里程碑，当前版本只包含“画”本身。

## 9. 技术栈

- NeoForge 1.21.1（Java 21）
- `BlockEntity` + `BlockEntityRenderer` + `DynamicTexture` 实时渲染
- `CustomPacketPayload`（vanilla 1.20.5+ 网络 API）做服务端/客户端同步
- 物品数据使用 `DataComponents.CUSTOM_DATA` 存储画作
